## 一、热学
### 1.1 理想气体基本公式
- 理想气体物态方程：$$pV=\frac{m}{M}RT$$
- 压强微观表达式：$$p = \frac{2}{3} n \overline{\varepsilon}_{\text{kt}} = \frac{2}{3} n \left( \frac{1}{2} m_0 \overline{v^2} \right)$$
- 物态方程另一形式：$$p = nkT$$
- 玻尔兹曼常量与气体常量关系：$$k=\frac{R}{N_{\text{A}}}$$

**符号说明：**
- **$p$**：压强（Pa）
- **$V$**：体积（m³）
- **$T$**：热力学温度（K）
- **$m$**：气体质量（kg）
- **$M$**：摩尔质量（kg/mol）
- **$n$**：分子数密度（m⁻³）
- **$R$**：气体常量（8.31 J/(mol·K)）
- **$k$**：玻尔兹曼常量（$1.38\times10^{-23}$ J/K）
- **$m_0$**：分子质量（kg）
- **$N_{\text{A}}$**：阿伏伽德罗常量（$6.02\times10^{23}$ mol⁻¹）

### 1.2 速率分布函数
- 定义：$$\frac{\mathrm{d}N}{N_0} = f(v)\mathrm{d}v \text{ 或 } f(v) = \frac{\mathrm{d}N}{N_0 \mathrm{d}v}$$
- 麦克斯韦速率分布：$$f(v) = 4\pi \left( \frac{m_0}{2\pi kT} \right)^{\frac{3}{2}} e^{-\frac{m_0 v^2}{2kT}} v^2$$
- 归一化条件：$$\int_{0}^{\infty} f(v) \mathrm{d}v = 1$$

**符号说明：**
- **$f(v)$**：速率分布函数（s/m）
- **$m_0$**：分子质量（kg）
- **$N_0$**：总分子数（个）

### 1.3 三种统计速率
| 速率类型 | 公式 | 物理意义 |
|----------|------|----------|
| 最概然速率 | $v_{\text{p}} = \sqrt{\frac{2kT}{m_0}} = \sqrt{\frac{2RT}{M}}$ | 分布函数最大值对应的速率 |
| 平均速率 | $\bar{v} = \sqrt{\frac{8kT}{\pi m_0}} = \sqrt{\frac{8RT}{\pi M}}$ | 速率的算术平均值 |
| 方均根速率 | $\sqrt{\overline{v^2}} = \sqrt{\frac{3kT}{m_0}} = \sqrt{\frac{3RT}{M}}$ | 速率平方平均的平方根 |

**符号说明：**
- **$v_{\text{p}}$**：最概然速率（m/s）
- **$\bar{v}$**：平均速率（m/s）
- **$\overline{v^2}$**：速率平方平均（m²/s²）

### 1.4 分子动能与自由度
- 平均平动动能：$$\overline{\varepsilon}_{\text{kt}} = \frac{1}{2} m_0 \overline{v^2}=\frac{3}{2}kT$$
- 自由度表格：

| 分子种类 | 平动自由度 | 转动自由度 | 总自由度$i$ |
|----------|------------|------------|-------------|
| 单原子分子 | 3 | 0 | 3 |
| 刚性双原子分子 | 3 | 2 | 5 |
| 刚性多原子分子 | 3 | 3 | 6 |

- 分子的平均动能：$$\overline{\varepsilon}_{\text{k}} = \frac{i}{2} kT$$（$i$为分子的自由度数）
- 理想气体内能：$$E=\frac{m}{M}\frac{i}{2}RT$$

**符号说明：**
- **$i$**：总自由度（无量纲）
- **$E$**：内能（J）
- **$\overline{\varepsilon}_{\text{k}}$**：分子平均动能（J）

### 1.5 平均自由程与碰撞频率
- 分子的碰撞截面：$$\sigma = \pi d^2$$
- 平均碰撞频率：$$\bar{z} = \sqrt{2} \sigma \bar{v} n = \sqrt{2} \pi d^2 \bar{v} n$$
- 平均自由程：$$\overline{\lambda} = \frac{\bar{v}}{\bar{z}} = \frac{1}{\sqrt{2} \sigma n} = \frac{kT}{\sqrt{2}\pi d^2p}$$

**符号说明：**
- **$\overline{\lambda}$**：平均自由程（m）
- **$\bar{z}$**：碰撞频率（Hz）
- **$d$**：分子有效直径（m）
- **$\sigma$**：碰撞截面（m²）
- **$n$**：分子数密度（m⁻³）

### 1.6 热力学过程
#### 1.6.1 热容相关
- 摩尔定容热容：$$C_{V,\text{m}} = \left( \frac{\mathrm{d}Q}{\mathrm{d}T} \right)_V = \frac{\mathrm{d}E}{\mathrm{d}T} = \frac{i}{2} R$$
- 摩尔定压热容：$$C_{p,\text{m}} = \left( \frac{\mathrm{d}Q}{\mathrm{d}T} \right)_p = \frac{\mathrm{d}E}{\mathrm{d}T} + R$$
- 迈耶公式：$$C_{p,\text{m}} = C_{V,\text{m}} + R$$
- 比热容比：$$\gamma = \frac{C_{p,\text{m}}}{C_{V,\text{m}}}$$

**符号说明：**
- **$C_{V,\text{m}}$**：摩尔定容热容（J/(mol·K)）
- **$C_{p,\text{m}}$**：摩尔定压热容（J/(mol·K)）
- **$\gamma$**：比热容比（无量纲）

#### 1.6.2 等值过程
| 过程类型 | 核心条件 | 做功 $W$ | 内能变化 $\Delta E$ | 热量 $Q$ |
|----------|----------|----------|---------------------|----------|
| 等体过程 | $V=$常量 | $0$ | $\frac{m}{M}C_{V,\text{m}}(T_2-T_1)$ | $Q_V = \Delta E$ |
| 等压过程 | $p=$常量 | $p(V_2-V_1)$ | $\frac{m}{M}C_{V,\text{m}}(T_2-T_1)$ | $Q_p = \Delta E + W$ |
| 等温过程 | $T=$常量 | $\frac{m}{M}RT\ln\frac{V_2}{V_1}$ | $0$ | $Q_T = W$ |
| 绝热过程 | $Q=0$ | $\frac{1}{\gamma-1}(p_1V_1-p_2V_2)$ | $-W$ | $0$ |

**补充公式：**
- 绝热过程泊松公式：$TV^{\gamma-1}=C_2$；$\frac{p^{\gamma-1}}{T^\gamma}=C_3$；$pV^\gamma=C_1$
- 绝热线斜率：$\left( \frac{\mathrm{d}p}{\mathrm{d}V} \right)_Q = -\gamma \frac{p}{V}$（等温线斜率：$\left( \frac{\mathrm{d}p}{\mathrm{d}V} \right)_T = -\frac{p}{V}$）

### 1.7 热机与熵
- 热机效率：$$\eta = \frac{W}{Q_1} = 1 - \frac{Q_2}{Q_1}$$
- 制冷系数：$$\varepsilon = \frac{Q_2}{W} = \frac{Q_2}{Q_1 - Q_2}$$
- 熵：$$S = k\ln \Omega$$
- 熵变：$$\Delta S = k\ln \frac{\Omega_2}{\Omega_1} > 0$$，$$\mathrm{d}S = \frac{\mathrm{d}Q}{T}$$
- 熵差计算：$$\Delta S = \int_{1}^{2} \mathrm{d}S \geqslant \int_{1}^{2} \frac{\mathrm{d}Q}{T}$$，$$\Delta S = \frac{m}{M} C_{V,\text{m}} \ln\frac{T_2}{T_1} + \frac{m}{M} R \ln\frac{V_2}{V_1}$$

**符号说明：**
- **$\eta$**：热机效率（无量纲）
- **$\varepsilon$**：制冷系数（无量纲）
- **$S$**：熵（J/K）
- **$\Omega$**：热力学概率（无量纲）
- **$Q_1$**：吸热（J）
- **$Q_2$**：放热（J）

### 1.8 例题参考
- 热学基础：3、5、6、13、18、20；
- 热力学过程：4、5、7、8、9、10、13、15、17、19、21；
- 教材：《大学物理学 (第五版)》王少杰

## 二、光学
### 2.1 半波损失
> **核心考点：半波损失**
- 定义：光从光疏→光密介质反射时，相位突变 $\pi$（光程差加 $\lambda/2$），折射光无半波损失；
- 条件：仅反射光、光疏→光密入射（$n_1 < n_2$）；
- 影响：干涉加强/减弱条件反转（薄膜/劈尖/牛顿环分析关键）。

**符号说明：**
- **$\lambda$**：入射光波长（m）
- **$n$**：介质折射率（无量纲）

### 2.2 光的干涉
#### 2.2.1 杨氏双缝
- 光程差：$$\delta = r_2 - r_1 \approx \frac{xd}{D}$$
- 明纹：$$\delta = \pm k\lambda \Rightarrow x = \pm k \frac{D\lambda}{d}\quad (k=0,1,2,\dots)$$
- 暗纹：$$\delta = \pm (2k-1)\frac{\lambda}{2} \Rightarrow x = \pm (2k-1)\frac{D\lambda}{2d}$$
- 条纹间距：$$\Delta x=\frac{D\lambda}{d}$$

**符号说明：**
- **$d$**：缝间距（m）
- **$D$**：屏距（m）
- **$x$**：条纹位置（m）
- **$k$**：条纹级次（无量纲）

#### 2.2.2 薄膜等倾干涉
- 光程差：$$\delta = 2e \sqrt{n_2^2 - n_1^2 \sin^2 i} + \frac{\lambda}{2}$$
- 明纹：$$2e \sqrt{n_2^2 - n_1^2 \sin^2 i} = \left( k - \frac{1}{2} \right)\lambda$$
- 暗纹：$$2e \sqrt{n_2^2 - n_1^2 \sin^2 i} = k\lambda$$

**符号说明：**
- **$e$**：薄膜厚度（m）
- **$n_1/n_2$**：介质折射率（无量纲）
- **$i$**：入射角（rad）

#### 2.2.3 劈尖干涉
| 类型 | 条件 | 公式 |
|------|------|------|
| 光程差 | 空气劈尖 | $\delta = 2e + \frac{\lambda}{2}$ |
| 明纹 | $2e + \frac{\lambda}{2} = k\lambda$ | $e = (2k-1)\frac{\lambda}{4}$ |
| 暗纹 | $2e + \frac{\lambda}{2} = (2k+1)\frac{\lambda}{2}$ | $e = k\frac{\lambda}{2}$ |
| 条纹间距 | 几何关系 | $L = \frac{\lambda}{2\theta}$ |

**符号说明：**
- **$e$**：空气层厚度（m）
- **$\theta$**：劈尖角（rad）

#### 2.2.4 牛顿环
- 光程差：$$\delta = 2e + \frac{\lambda}{2}$$（$e \approx \frac{r^2}{2R}$）
- 明环：$$r_k = \sqrt{(k-\frac{1}{2})\lambda R}$$
- 暗环：$$r_k = \sqrt{k\lambda R}$$
- 曲率半径测量：$$R=\frac{d_{k+m}^{2}-d_{k}^{2}}{4 m \lambda}$$

**符号说明：**
- **$r_k$**：环半径（m）
- **$R$**：透镜曲率半径（m）
- **$d_k$**：第$k$圈暗环直径（m）

### 2.3 光的衍射
#### 2.3.1 单缝衍射（夫琅禾费）
> **核心考点：单缝衍射**
- 半波带法：最大光程差 $\Delta = a\sin\theta$，半波带数 $N = \frac{2a\sin\theta}{\lambda}$
- 暗纹：$$a\sin\theta = \pm k\lambda\quad (k=1,2,3,\dots)$$
- 中央明纹：半角宽度 $\theta_0 \approx \frac{\lambda}{a}$，角宽度 $\Delta\theta_0 \approx \frac{2\lambda}{a}$
- 相对光强：$$\frac{I}{I_0} = \left( \frac{\sin\alpha}{\alpha} \right)^2$$（$\alpha = \frac{\pi a\sin\theta}{\lambda}$）

**符号说明：**
- **$a$**：缝宽（m）
- **$\theta$**：衍射角（rad）
- **$\Delta\theta_0$**：中央明纹角宽度（rad）

#### 2.3.2 光栅衍射
- 光栅常量：$d=a+b$（$a$ 缝宽，$b$ 刻痕宽）
- 光栅方程（主极大）：$$d\sin\varphi = \pm k\lambda\quad (k=0,1,2,\dots)$$
- 缺级条件：$$k = k'\frac{d}{a}\quad (k'=\pm1,\pm2,\dots)$$

**符号说明：**
- **$d$**：光栅常量（m）
- **$\varphi$**：衍射角（rad）
- **$k$**：主极大级次（无量纲）

#### 2.3.3 光学仪器的分辨本领
- 瑞利判据：$$\delta\theta = 1.22 \frac{\lambda}{d}$$
- 分辨本领：$$R = \frac{1}{\delta\theta} = \frac{d}{1.22\lambda}$$

**符号说明：**
- **$\lambda$**：入射光波长（m）
- **$d$**：透镜/圆孔直径（m）

### 2.4 光的偏振
#### 2.4.1 马吕斯定律
$$I_2 = I_1 \cos^2\alpha$$（自然光入射：$I_1 = \frac{I_0}{2}$）

**符号说明：**
- **$I_0$**：入射光强（W/m²）
- **$\alpha$**：偏振方向夹角（rad）

#### 2.4.2 布儒斯特定律
$$\tan i_0 = \frac{n_2}{n_1}$$

**补充说明：**
- 起偏角$i_0$下，反射光为线偏振光（垂直入射面），反射光$\perp$折射光；
- 玻璃片堆多次反射后折射光为线偏振光。

#### 2.4.3 波片
| 波片类型 | 相位差 | 核心效果 |
|----------|--------|----------|
| 1/4波片 | $\Delta\varphi = (2k+1)\frac{\pi}{2}$ | 线偏振光（45°入射）→ 圆偏振光 |
| 1/2波片 | $\Delta\varphi = (2k+1)\pi$ | 线偏振光振动面转过 $2\alpha$ |

### 2.5 例题参考
- 光的干涉：8、9、11、13、16、18、19、20；
- 光的衍射：23、24、25、26、28；
- 光的偏振：32、33、36、38、40；
- 教材：《大学物理学 (第五版)》王少杰

## 三、量子物理
### 3.1 早期量子论
#### 3.1.1 黑体辐射
- 斯特藩-玻尔兹曼定律：$$M_0(T) = \sigma_0 T^4$$
- 维恩位移定律：$$\lambda_{\text{m}}T=b$$

**常数：**
- $\sigma_0=5.67\times10^{-8}$ W/(m²·K⁴)
- $b=2.898\times10^{-3}$ m·K

#### 3.1.2 光电效应
$$h\nu = \frac{1}{2}mv_{\text{m}}^2 + A_0$$（光子能量：$\varepsilon = h\nu$）

**符号说明：**
- **$h$**：普朗克常量（$6.63\times10^{-34}$ J·s）
- **$\nu$**：频率（Hz）
- **$A_0$**：逸出功（J）
- **$v_{\text{m}}$**：光电子最大速度（m/s）

#### 3.1.3 康普顿效应
- 波长改变量：$$\Delta\lambda = \frac{h}{m_0 c}(1-\cos\theta) = \frac{2h}{m_0 c}\sin^2\frac{\theta}{2}$$
- 康普顿波长：$$\lambda_{\text{C}}=\frac{h}{m_0c}=2.43\times10^{-12}$$ m

**符号说明：**
- **$m_0$**：电子静质量（$9.11\times10^{-31}$ kg）
- **$c$**：光速（$3\times10^8$ m/s）
- **$\theta$**：散射角（rad）

#### 3.1.4 德布罗意波
- 物质波波长（非相对论）：$$\lambda = \frac{h}{p} = \frac{h}{\sqrt{2m_0E_{\text{k}}}}$$
- 物质波波长（相对论）：$$\lambda = \frac{hc}{\sqrt{E_{\text{k}}^2 + 2m_0c^2E_{\text{k}}}}$$
- 玻尔轨道量子化：$$2\pi r = n\lambda \Rightarrow mvr = n\hbar$$

**符号说明：**
- **$p$**：动量（kg·m/s）
- **$\lambda$**：物质波波长（m）
- **$\hbar$**：约化普朗克常量（$1.05\times10^{-34}$ J·s）
- **$E_{\text{k}}$**：动能（J）

### 3.2 量子力学基础
#### 3.2.1 波函数
> **核心考点：波函数核心性质**
- 概率密度：$$w = |\Psi|^2 = \Psi^* \cdot \Psi$$
- 归一化条件：$$\iiint_{-\infty}^{+\infty} |\Psi|^2 \mathrm{d}x\mathrm{d}y\mathrm{d}z = 1$$
- 标准条件：单值、有限、连续

#### 3.2.2 不确定关系
> **核心考点：不确定性原理核心应用**
- 位置-动量不确定关系：$$\Delta x \Delta p_x \ge \frac{\hbar}{2}$$
- 时间-能量不确定关系：$$\Delta E \cdot \Delta t \ge \frac{\hbar}{2}$$
- 计算技巧：结合德布罗意关系 $p=\frac{h}{\lambda}$，由动量相对不确定度求位置不确定度

#### 3.2.3 薛定谔方程与一维势阱
> **核心考点：一维无限深势阱核心考点（含宽度2a特殊情形）**
- 一维定态薛定谔方程：$$-\frac{\hbar^2}{2m}\frac{\mathrm{d}^2\psi}{\mathrm{d}x^2} + V(x)\psi = E\psi$$
- 宽度为$a$的一维无限深势阱（$0 < x < a$）：
  - 归一化波函数：$$\psi_n(x) = \sqrt{\frac{2}{a}} \sin\left(\frac{n\pi x}{a}\right)$$
  - 能量本征值：$$E_n=\frac{n^2h^2}{8ma^2}\quad (n=1,2,3,\dots)$$
- 宽度为$2a$的一维无限深势阱（$0 < x < 2a$）：
  - 归一化波函数：$$\psi_n(x) = \begin{cases} \sqrt{\dfrac{1}{a}} \sin\left(\dfrac{n\pi x}{2a}\right), & 0 < x < 2a \\ 0, & x \le 0 \text{ 或 } x \ge 2a \end{cases}$$
  - 能量本征值：$$E_n = \frac{n^2 \pi^2 \hbar^2}{8 m a^2} = \frac{n^2 h^2}{32 m a^2}\quad (n=1,2,3,\dots)$$
- 能级跃迁与光子频率：
  - 第二激发态（$n=3$）→ 基态（$n=1$）能量差：$$\Delta E = E_3 - E_1 = \frac{\pi^2 \hbar^2}{m a^2}$$
  - 辐射光子频率：$$\nu = \frac{\Delta E}{h} = \frac{\pi \hbar}{2 m a^2}$$
- 相邻能级差与经典极限：
  - 相邻能级差：$$\Delta E_n = E_{n+1} - E_n = \frac{(2n+1)\pi^2 \hbar^2}{8 m a^2}$$
  - 经典极限：$n \to \infty$时，$\frac{\Delta E_n}{E_n} = \frac{2n+1}{n^2} \to 0$，能级连续，量子结果趋近经典理论

**符号说明：**
- **$m$**：粒子质量（kg）
- **$\hbar$**：约化普朗克常量（$\hbar = h/(2\pi)$）
- **$n$**：量子数（无量纲，$n=1,2,3,\dots$）
- **$E_n$**：第$n$能级能量（J）

#### 3.2.4 氢原子量子数
> **核心考点：氢原子量子数核心考点**
- 量子态存在条件：$n\ge1,\ 0\le l \le n-1,\ -l\le m_l\le l,\ m_s=\pm\frac{1}{2}$
- 角动量计算：$$L=\sqrt{l(l+1)}\hbar$$，$$L_z=m_l\hbar$$
- 轨道对应：$l=0(\text{s}),1(\text{p}),2(\text{d})$，如$3\text{p}$对应$n=3,l=1$

| 量子数 | 取值范围 | 物理意义 |
|--------|----------|----------|
| 主量子数 $n$ | $n=1,2,3,\dots$ | 决定能量：$E_n = -\frac{13.6}{n^2}$ eV |
| 角量子数 $l$ | $l=0,1,\dots,n-1$ | 决定角动量：$L=\sqrt{l(l+1)}\hbar$ |
| 磁量子数 $m_l$ | $m_l=0,\pm1,\dots,\pm l$ | 决定角动量分量：$L_z=m_l\hbar$ |
| 自旋磁量子数 $m_s$ | $m_s=\pm\frac{1}{2}$ | 决定自旋角动量：$S_z=m_s\hbar$ |
